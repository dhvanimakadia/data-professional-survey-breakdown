# Data Professional Survey Breakdown

## Description
This Power BI project visualizes insights from a survey of data professionals. It provides an interactive dashboard to explore trends, patterns, and insights related to roles, salaries, satisfaction, and work-life balance.

## Key Insights
- Average salary by job role
- Work-life balance analysis
- Salary satisfaction overview
- Country-wise responses
- Programming language preferences
- Difficulty in entering the data field

## Tools Used
- Power BI Desktop

## Outcome
An interactive dashboard to support decision-making and insights for data professionals.
